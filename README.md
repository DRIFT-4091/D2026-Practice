# D2026-Practice

This repository contains the code from D2026-Regional for practice purposes.

## Description
Current code for the practice Swerve drivetrain, copied from the D2026-Regional repository.

## Building and Deploying
This is a standard WPILib robot project. Use the following commands:
- `./gradlew build` - Build the project
- `./gradlew deploy` - Deploy to the robot

## Project Structure
- `src/main/java/frc/robot/` - Main robot code
- `vendordeps/` - Vendor dependencies (Phoenix6, REVLib, PathplannerLib)
- `build.gradle` - Gradle build configuration
- `.wpilib/` - WPILib configuration
